Seletores CSS
Selectores CSS Usado Onde
<UNIVERSAL> O seletor universal (*) aplica o estilo a todos os elementos da página
<TIPO> O seletor de tipo usa o nome do marcador a estilizar
<CLASSE> O seletor de classe associa um nome a um estilo
<ID> O atributo id permite identificar univocamente um elemento do HTML
<ATRIBUTO> Seletores que se aplicam a atributos de elementos
<PSEUDO-CLASSE> Seletor associado ao estado de um elemento
<PSEUDO-ELEMENTO> Seletor associado em partes de um elemento




Combinadores CSS
Combinadores de Seletores CSS Usado Onde
<DESCENDENTES> Todos os tag que estão “dentro” de um tag
<FILHOS> Todos os tag que estão “logo dentro" de tag
<ADJACENTES> Todos os tag que estão colocados depois de um tag
<IMEDIATAMENTE ADJACENTES> Todos os tag que estão colocados “imediatamente” depois de um tag
<AGRUPADOS> Mudar todos os tag ao mesmo tempo
